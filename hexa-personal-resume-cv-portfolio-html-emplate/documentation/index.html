Template Installation

You can view this template in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the template from your text editor.

All of the HTML files are given a descriptive name to make it easier to distingish what each file will display.

FTP Upload:

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Open Hexa folder and you will find 2 subfolder name css-version(If you can't use sass) & sass-version(If you can use sass)
  • Upload the files inside Hexa folder of one of the 2 subfolder.

Customization

Change Color:

You can easily change theme color.
For sass user open your project file in a sass complier (Prepros App, Koala, Compass.app) and open _variables.scss file and edit the folowing code and save it .

// Theme Color
$themeColorOne: #3f51b5;
$themeColorTwo: #ff4081;

For css user open your project file in a text editor and open style.css file and find the folowing line of code. Here is both theme color. Now you have to replace those two color for the hole file. You can to it by pressing the Ctrl+H button in keybord

.overlay::before {
    background: #3f51b5;
    background: transparent -webkit-linear-gradient(left top, #3f51b5, #ff4081);
    background: transparent linear-gradient(to right bottom, #3f51b5, #ff4081);
    bottom: 0;
    content: "";
    left: 0;
    opacity: 0.85;
    position: absolute;
    right: 0;
    top: 0;
}

Change Logo:

  • Open the .html file (example index.html).
  • Find this line of code in <div class="header-area animated"> section:
<a class="logo" href="index.html">hexa</a>
  • I use text, you can use text for your logo or use a image by using <img src="image link here"/> tag.

Change Overlay to Solid Background:

  • Open the .html file (example index.html).
  • Find section With overlay Class and add a new class solid.

Change Slideshow images:

  • Open the .html file (example index-kenburne-slideshow.html).
  • Find Intro section.
  • In intro section you wiil find kenburne-slider and under this class you will find img tag with image that you can change, Remove or add a new image.
<!--Kenburne Slideshow-->
<div class="kenburne-slider">
    <img src="img/section-bg/6.jpg" alt="slideshow" />
    <img src="img/section-bg/7.jpg" alt="slideshow" />
</div>

Change HTML5 Background Video:

  • Open the main.js file
  • Find this line of code
$bgSelfHostVideo.vide({
    'mp4': 'video/shutter-island',
    'poster': 'video/poster',
});
  • Change those video to your video url
  • You will need video in mp4 format and a banner image in jpg format

Change Youtube Background Video:

  • Open the .html file (example index-youtube-video.html).
  • Find Intro section with attribute data-property in it you will find videoURL change it value with your youtube video id.
videoURL:'eZ70TaAUhQo'

Change Vimeo Background Video:

  • Open the .html file (example index-vimeo-video.html).
  • Find Intro section with attribute data-property in it you will find videoURL change it value with your vimeo video url.
videoURL:'https://vimeo.com/34234286'
